# Python Control Structures Assignment

This repository contains solutions for two Python programming tasks focusing on control structures (if-else statements and loops).

## Tasks Overview

### Task 1: Check if a Number is Even or Odd
**File:** `task1_even_odd.py`

**Description:**
A Python program that:
1. Takes an integer input from the user
2. Checks whether the number is even or odd using an if-else statement
3. Displays the result accordingly

**Features:**
- Input validation for integers
- Error handling for invalid inputs
- Continuous operation until user chooses to exit
- Clean, formatted output

**How to Run:**
```bash
python task1_even_odd.py
```
###### Example Output:
```text
Task 1: Even or Odd Number Checker
========================================
Enter a number: 7

========================================
7 is an odd number.
========================================

Do you want to check another number? (yes/no): yes
========================================
Enter a number: 12

========================================
12 is an even number.
========================================
```
###### Task 2: Sum of Integers from 1 to 50 Using a Loop
File: task2_sum_numbers.py

###### Description:
A Python program that:
- Uses a for loop to iterate over numbers from 1 to 50
- Calculates the sum of all integers in this range
- Displays the final sum

###### Features:
- Multiple calculation methods (for loop, while loop, mathematical formula)
- Result verification
- Optional advanced version with custom range
- Detailed output with explanations
